Navigating the Main Window

Figure 1 shows the main window prior to a CDD file being loaded. There are six main components of the main window. These components are the following (from top to bottom, left to right):

  1. Main Menu Bar
  2. Coverage Type View Bar
  3. Summary Coverage Bar
  4. Module/Instance Listbox
  5. Coverage File Viewer
  6. Information Bar
