The Main Menu Bar

The Main Menu Bar consists of four menus at the top of the main window. Figure 1. below shows the menu bar portion of the GUI bordered in red. These menus are the following:

  • File - Opens new CDD files for viewing and exits the application
  • Report - Specifies what type of report to show
  • Preferences - Allows users to change GUI attributes pertaining to the coverage information
  • Help - Provides access to online help manual and tool information
